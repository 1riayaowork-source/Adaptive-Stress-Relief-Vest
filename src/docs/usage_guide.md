# 使用指南
## 1. 硬件接线检查
- 确认所有GND共地
- 确认ESP32 5V接Arduino 5V，而非6V锂电池
- 确认气泵正极接MOS模块OUT/SIG，负极接锂电池GND

## 2. 代码上传后验证
### 2.1 串口监视器检查
打开Arduino IDE串口监视器（波特率115200），应看到：
连接 WiFi：xxx
....
WiFi 连接成功！
虚拟控制器地址：192.168.x.x
Web 服务器已启动！

### 2.2 OLED显示检查
OLED屏应显示：
Air Vest Controller
Pressure: xx%
State: Idle


## 3. 网页控制器使用
- 在手机/电脑浏览器输入串口显示的IP地址
- 点击“压力归位”：气泵自动工作，直到压力达到目标值
- 点击“放气”：放气阀打开（如有），气泵关闭
- 网页会实时显示当前状态

## 4. 常见问题排查
### 4.1 网页打不开
- 确认设备和ESP32连同一WiFi
- 确认输入的IP是串口显示的真实IP
- 关闭手机/电脑的流量（避免走移动网络）

### 4.2 气泵不工作
- 检查MOS模块VCC是否接5V
- 检查ESP32 GPIO26是否输出HIGH
- 检查气泵是否接6V锂电池

### 4.3 OLED不显示
- 修改OLED地址（0x3C ↔ 0x3D）
- 检查OLED SDA/SCL是否接ESP32 21/22
