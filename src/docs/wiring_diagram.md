# 接线示意图（文字版）
## 核心原则
所有硬件GND必须共地！

## 1. 供电部分
| 设备         | 供电来源                  | 备注                          |
|--------------|---------------------------|-------------------------------|
| ESP32        | Arduino 5V引脚            | 避免6V直接接ESP32             |
| MOS模块      | Arduino 5V引脚            | 驱动能力更强                  |
| OLED屏       | ESP32 3.3V引脚            | 安全供电                      |
| 6V气泵       | 6V磷酸铁锂电池            | 经MOS模块OUT/SIG引脚控制      |

## 2. 控制引脚
| ESP32引脚 | 连接对象                | 功能               |
|-----------|-------------------------|--------------------|
| GPIO26    | MOS模块印“SCL”的引脚    | 控制气泵           |
| GPIO27    | MOS模块印“SDA”的引脚    | 控制放气阀（无则空）|
| GPIO21    | OLED SDA                | OLED数据           |
| GPIO22    | OLED SCL                | OLED时钟           |
| GND       | 所有硬件GND             | 共地               |

## 3. MOS模块接线
| MOS模块引脚 | 连接对象                | 功能               |
|-------------|-------------------------|--------------------|
| VCC         | Arduino 5V              | 供电               |
| GND         | 总地线                  | 共地               |
| OUT/SIG     | 6V气泵正极              | 气泵电源输出       |
| 气泵负极    | 6V锂电池GND             | 接地               |
