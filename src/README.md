# 充气马甲控制器（ESP32 + Web控制 + OLED显示）
基于ESP32的充气马甲智能控制器，支持网页虚拟按钮手动控制气泵/放气阀，OLED实时显示压力和设备状态，适配6V气泵和磷酸铁锂电池供电。

## 功能特点
- 📱 网页虚拟控制器：浏览器访问ESP32 IP，点击按钮控制“压力归位”/“放气”
- 🖥️ OLED显示：实时显示当前压力值、设备工作状态（空闲/放气中/归位中）
- ⚡ 硬件适配：完美兼容6V气泵、IIC MOS模块、0.96寸OLED、6V磷酸铁锂电池
- 🛠️ 自动归位：压力达到目标值后自动停止，无需手动干预

## 硬件清单
| 硬件名称                | 规格/型号                  | 备注                          |
|-------------------------|----------------------------|-------------------------------|
| ESP32 DevKitC           | 任意款                     | 核心控制器                    |
| MOS模块                 | 4引脚（GND/VCC/SCL/SDA）   | 标注SCL/SDA实际为普通GPIO     |
| 6V微型气泵              | PYP370-6A                  | 充气动力                      |
| 0.96寸OLED屏            | IIC接口                    | 状态显示                      |
| 6V磷酸铁锂电池组        | 1800mAh                    | 供电                          |
| Arduino主板             | 任意款                     | 给ESP32供5V（替代降压模块）   |
| 杜邦线                  | 公对公/公对母              | 接线                          |

## 快速开始
### 1. 安装依赖库
在Arduino IDE中安装以下库：
- `WiFi`（ESP32自带）
- `WebServer`（ESP32自带）
- `Wire`（ESP32自带）
- `SSD1306`（ESP32 OLED驱动）

### 2. 修改配置
打开 `src/air_vest_controller.ino`，修改：
- WiFi名称和密码
- （可选）OLED地址（0x3C/0x3D）
- （可选）压力阈值、引脚定义

### 3. 上传代码
- 将ESP32连接电脑，选择对应开发板和端口
- 上传代码到ESP32

### 4. 访问控制器
- 打开串口监视器（波特率115200），查看ESP32的IP地址
- 在浏览器中输入该IP，即可看到虚拟控制器界面
